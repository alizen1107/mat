1. KOMPLEKS ƏDƏDLƏR VƏ ONLAR ÜZƏRİNDƏ ƏMƏLLƏR

1.1. Kompleks Ədədlərin Tərifi

Kompleks ədədlər riyaziyyatın ən mühüm anlayışlarından biridir. Tarixi olaraq, kompleks ədədlər XVI əsrdə kubik tənliklərin həllində meydana çıxmışdır. İtalyan riyaziyyatçıları Cardano və Bombelli bu sahədə ilk addımları atmışlar.

Kompleks ədəd z aşağıdakı formada yazılır:

z = a + bi

Burada:

  • a - həqiqi hissə (Re(z) = a)
  • b - xəyali hissə (Im(z) = b)
  • i - xəyali vahid (i² = -1)
Qeyd: Xəyali vahid i ədədi i² = -1 şərtini ödəyir. Bu, həqiqi ədədlər çoxluğunda həll olunmayan x² + 1 = 0 tənliyinin həllidir.

1.2. Kompleks Müstəvisi

Kompleks ədədləri həndəsi olaraq kompleks müstəvisində (Gauss müstəvisi) təsvir etmək olar. Bu müstəvidə:

  • Üfüqi ox - həqiqi ox (Re)
  • Şaquli ox - xəyali ox (Im)
  • Hər kompleks ədəd z = a + bi müstəvidə (a, b) nöqtəsi kimi təsvir olunur

1.3. Kompleks Ədədin Modulü və Arqumenti

Kompleks ədədin modulu (və ya mütləq qiyməti) onun başlanğıc nöqtəsindən məsafəsidir:

|z| = √(a² + b²)

Kompleks ədədin arqumenti (və ya fazası) müsbət həqiqi oxla z vektorunun əmələ gətirdiyi bucaqdır:

arg(z) = φ = arctan(b/a)
Nümunə 1.1: z = 3 + 4i kompleks ədədinin modulunu və arqumentini tapaq.

Həll:
|z| = √(3² + 4²) = √(9 + 16) = √25 = 5
arg(z) = arctan(4/3) ≈ 53.13°

1.4. Kompleks Ədədin Triqqonometrik və Eksponensial Formaları

Kompleks ədədi müxtəlif formalarda yazmaq olar:

Triqqonometrik forma:

z = r(cos φ + i sin φ)

Burada r = |z| - modul, φ = arg(z) - arqumentdir.

Eksponensial forma (Eyler düsturu):

z = re^(iφ)

Eyler düsturu:

e^(iφ) = cos φ + i sin φ
Eyler düsturunun xüsusi halları:
e^(iπ) + 1 = 0 (Eyler eynilik)
e^(i·π/2) = i
e^(i·2π) = 1

1.5. Kompleks Ədədlər Üzərində Əməllər

1.5.1. Toplama və Çıxma

İki kompleks ədədin toplanması və çıxılması həqiqi və xəyali hissələrin ayrıca toplanması/çıxılması ilə həyata keçirilir:

z₁ + z₂ = (a₁ + a₂) + (b₁ + b₂)i
z₁ - z₂ = (a₁ - a₂) + (b₁ - b₂)i
Nümunə 1.2: z₁ = 2 + 3i və z₂ = 1 - 2i ədədlərini toplayaq və çıxaq.

Həll:
z₁ + z₂ = (2 + 1) + (3 - 2)i = 3 + i
z₁ - z₂ = (2 - 1) + (3 - (-2))i = 1 + 5i

1.5.2. Vurma

Alqebraik formada vurma:

z₁ · z₂ = (a₁ + b₁i)(a₂ + b₂i) = (a₁a₂ - b₁b₂) + (a₁b₂ + a₂b₁)i

Triqqonometrik formada vurma (daha sadə):

z₁ · z₂ = r₁r₂[cos(φ₁ + φ₂) + i sin(φ₁ + φ₂)]
Qeyd: Kompleks ədədləri vurarkən modullar vurulur, arqumentlər isə toplanır.
Nümunə 1.3: z₁ = 1 + i və z₂ = 2 - i ədədlərini vurun.

Həll:
z₁ · z₂ = (1 + i)(2 - i) = 2 - i + 2i - i² = 2 + i - (-1) = 3 + i

1.5.3. Bölmə

Kompleks ədədlərin bölünməsi üçün kəsri məxrəcini həqiqi ədədə çevirmək lazımdır. Bunun üçün kəsrin həm surətini, həm də məxrəcini məxrəcin qoşma kompleks ədədinə vururuq:

z₁/z₂ = (a₁ + b₁i)/(a₂ + b₂i) = [(a₁ + b₁i)(a₂ - b₂i)]/[(a₂ + b₂i)(a₂ - b₂i)]
= [(a₁a₂ + b₁b₂) + (a₂b₁ - a₁b₂)i]/(a₂² + b₂²)

Triqqonometrik formada bölmə:

z₁/z₂ = (r₁/r₂)[cos(φ₁ - φ₂) + i sin(φ₁ - φ₂)]
Nümunə 1.4: (3 + 4i)/(1 + 2i) bölməsini yerinə yetirək.

Həll:
(3 + 4i)/(1 + 2i) = [(3 + 4i)(1 - 2i)]/[(1 + 2i)(1 - 2i)]
= [3 - 6i + 4i - 8i²]/[1 - 4i²]
= [3 - 2i + 8]/[1 + 4]
= (11 - 2i)/5 = 11/5 - 2i/5 = 2.2 - 0.4i

1.6. Qoşma Kompleks Ədəd

z = a + bi kompleks ədədinin qoşma kompleks ədədi z̄ (və ya z*) aşağıdakı kimi təyin olunur:

z̄ = a - bi

Qoşma kompleks ədədin xassələri:

  • z + z̄ = 2a = 2Re(z)
  • z - z̄ = 2bi = 2i·Im(z)
  • z · z̄ = a² + b² = |z|²
  • (z̄)̄ = z
  • (z₁ + z₂)̄ = z̄₁ + z̄₂
  • (z₁ · z₂)̄ = z̄₁ · z̄₂

1.7. Kompleks Ədədin Qüvvətə Yüksəldilməsi

De Moivre düsturu:

z^n = r^n(cos(nφ) + i sin(nφ)) = r^n · e^(inφ)
Nümunə 1.5: (1 + i)⁴ hesablayaq.

Həll:
Əvvəlcə triqqonometrik formaya keçək:
r = √(1² + 1²) = √2
φ = arctan(1/1) = π/4
z = √2(cos(π/4) + i sin(π/4))

De Moivre düsturuna görə:
z⁴ = (√2)⁴(cos(4·π/4) + i sin(4·π/4))
= 4(cos π + i sin π)
= 4(-1 + 0i) = -4

1.8. Kompleks Ədəddən Kök Çıxarma

z kompleks ədədindən n-ci dərəcədən kök çıxarmaq üçün aşağıdakı düsturdan istifadə olunur:

ⁿ√z = ⁿ√r[cos((φ + 2πk)/n) + i sin((φ + 2πk)/n)]

Burada k = 0, 1, 2, ..., n-1 və n fərqli kök alınır.

Nümunə 1.6: 1-dən kubik kök çıxaraq (³√1).

Həll:
1 = 1(cos 0 + i sin 0)
r = 1, φ = 0

k = 0: ³√1 = cos(0/3) + i sin(0/3) = 1
k = 1: ³√1 = cos(2π/3) + i sin(2π/3) = -1/2 + i√3/2
k = 2: ³√1 = cos(4π/3) + i sin(4π/3) = -1/2 - i√3/2

1.9. Kompleks Ədədlərin Tətbiqləri

Kompleks ədədlər riyaziyyatın və fizikanın bir çox sahələrində tətbiq olunur:

  • Elektrotexnika: Dəyişən cərəyan dövrələrinin təhlili
  • Kvant mexanikası: Dalğa funksiyalarının təsviri
  • Siqnal emalı: Furye çevrilməsi
  • Aerodinamika: Maye axınının modelləşdirilməsi
  • Fraktallar: Mandelbrot çoxluğu
  • Rəqəmsal görüntü emalı: Filtrləmə və sıxılma

1.10. Kompleks Funksiyalar

Kompleks dəyişənli funksiya f: ℂ → ℂ şəklində yazılır. Ən çox istifadə olunan kompleks funksiyalar:

Eksponensial funksiya:

e^z = e^(a+bi) = e^a(cos b + i sin b)

Triqqonometrik funksiyalar:

sin z = (e^(iz) - e^(-iz))/(2i)
cos z = (e^(iz) + e^(-iz))/2

Loqarifmik funksiya:

ln z = ln|z| + i·arg(z) + 2πki, k ∈ ℤ
Qeyd: Kompleks loqarifm çoxqiymətli funksiyadır, çünki arqument 2π-yə qədər dəqiqliklə müəyyən olunur.

2. BİR TƏRTİBLİ XƏTTİ DİFERENSİAL TƏNLİK. BERNOULLİ TƏNLİYİ

2.1. Diferensial Tənliklərin Tərifi

Diferensial tənlik naməlum funksiyanı, onun törəmələrini və müstəqil dəyişəni əlaqələndirən tənlikdir. Bir tərtibli diferensial tənlik ümumi formada belə yazılır:

F(x, y, y') = 0

2.2. Bir Tərtibli Xətti Diferensial Tənlik

Bir tərtibli xətti diferensial tənlik aşağıdakı formadadır:

y' + P(x)y = Q(x)

Bu tənliyin həlli üçün inteqrallayıcı vuruq metodundan istifadə olunur.

Həll metodu:

  1. İnteqrallayıcı vuruq μ(x) = e^(∫P(x)dx) tapırıq
  2. Tənliyin hər iki tərəfini μ(x)-ə vururuq
  3. Alınan tənliyi inteqrallayırıq
Nümunə 2.1: y' + 2y = 4x tənliyini həll edək.

Həll:
P(x) = 2, Q(x) = 4x
μ(x) = e^(∫2dx) = e^(2x)
e^(2x)y' + 2e^(2x)y = 4xe^(2x)
(e^(2x)y)' = 4xe^(2x)
e^(2x)y = ∫4xe^(2x)dx = 2xe^(2x) - e^(2x) + C
y = 2x - 1 + Ce^(-2x)

2.3. Bernoulli Tənliyi

Bernoulli tənliyi aşağıdakı formadadır:

y' + P(x)y = Q(x)y^n

Burada n ≠ 0 və n ≠ 1. Bu tənlik xətti deyil, lakin xətti tənliyə gətirilə bilər.

Həll metodu:

  1. z = y^(1-n) əvəzləməsi aparırıq
  2. z' = (1-n)y^(-n)y' alırıq
  3. Xətti tənlik alırıq: z' + (1-n)P(x)z = (1-n)Q(x)
  4. Xətti tənlik kimi həll edirik
Nümunə 2.2: y' + y = xy² tənliyini həll edək.

Həll:
n = 2, z = y^(-1) = 1/y
z' = -y^(-2)y'
-z' + z = x
z' - z = -x
μ(x) = e^(-x)
Həll: z = -x - 1 + Ce^x
y = 1/(-x - 1 + Ce^x)

2.4. Dəyişənlərə Ayrılan Tənliklər

Dəyişənlərə ayrılan tənlik forması:

M(x)dx + N(y)dy = 0

Həll: ∫M(x)dx + ∫N(y)dy = C

3. DETERMİNANTIN XASSƏLƏRİ. TƏRS MATRİS. MATRİSİN RANQI

3.1. Determinant Anlayışı

Determinant kvadrat matrisə uyğun ədədi xarakteristikadır. 2×2 matris üçün:

det(A) = |a b| = ad - bc
         |c d|

3.2. Determinantın Xassələri

  1. det(A^T) = det(A)
  2. İki sətir (sütun) dəyişdirildikdə determinant işarəsini dəyişir
  3. İki eyni sətir (sütun) varsa, det(A) = 0
  4. Sətir (sütun) ədədə vurulsa, determinant həmin ədədə vurulur
  5. det(AB) = det(A)·det(B)
  6. det(A^(-1)) = 1/det(A)

3.3. Tərs Matris

A matrisi üçün tərs matris A^(-1) aşağıdakı şərti ödəyir:

A·A^(-1) = A^(-1)·A = I

Tərs matrisin mövcudluq şərti: det(A) ≠ 0

2×2 matris üçün tərs matris:

A^(-1) = (1/det(A))·| d -b|
                      |-c a|
Nümunə 3.1: A = |2 1| matrisinin tərsini tapaq.
                               |3 4|

Həll:
det(A) = 2·4 - 1·3 = 5
A^(-1) = (1/5)·| 4 -1| = | 0.8 -0.2|
                  |-3 2|   |-0.6 0.4|

3.4. Matrisin Ranqı

Matrisin ranqı onun xətti asılı olmayan sətir (və ya sütun) sayıdır. Rank(A) ilə işarə olunur.

Ranqın xassələri:

  • 0 ≤ Rank(A) ≤ min(m,n)
  • Rank(A) = Rank(A^T)
  • Rank(AB) ≤ min(Rank(A), Rank(B))

Ranqın hesablanması:

  1. Qauss metodu ilə pilləli formaya gətirmək
  2. Sıfır olmayan sətirlərin sayını saymaq
Nümunə 3.2: Matrisin ranqını tapaq:
A = |1 2 3|
     |2 4 6|
     |1 1 1|

Həll: İkinci sətir birinci sətirin 2 qatıdır.
Pilləli forma: |1 2 3|
                    |0 -3 -5|
                    |0 0 0|
Rank(A) = 2

4. İKİ TƏRTİBLİ ƏYRİLƏR

4.1. Çevrə

Çevrə müstəvidə sabit nöqtədən (mərkəz) eyni məsafədə olan nöqtələr çoxluğudur.

(x - a)² + (y - b)² = r²

Burada (a,b) - mərkəz, r - radius

4.2. Ellips

Ellips müstəvidə iki sabit nöqtədən (fokus) məsafələrinin cəmi sabit olan nöqtələr çoxluğudur.

x²/a² + y²/b² = 1

Burada:

  • a - böyük yarımox
  • b - kiçik yarımox
  • c = √(a² - b²) - fokus məsafəsi
  • e = c/a - ekssentrikllik (0 < e < 1)
Nümunə 4.1: x²/25 + y²/16 = 1 ellipsinin parametrlərini tapaq.

Həll:
a² = 25, a = 5
b² = 16, b = 4
c = √(25-16) = 3
Fokuslar: F₁(-3,0), F₂(3,0)
e = 3/5 = 0.6

4.3. Hiperbola

Hiperbola müstəvidə iki sabit nöqtədən (fokus) məsafələrinin fərqi sabit olan nöqtələr çoxluğudur.

x²/a² - y²/b² = 1

Burada:

  • a - həqiqi yarımox
  • b - xəyali yarımox
  • c = √(a² + b²) - fokus məsafəsi
  • e = c/a - ekssentrikllik (e > 1)

Asimptotlar:

y = ±(b/a)x

4.4. Parabola

Parabola müstəvidə sabit nöqtədən (fokus) və sabit düz xəttdən (direktrisa) eyni məsafədə olan nöqtələr çoxluğudur.

y² = 2px

Burada:

  • p - parametr
  • Fokus: F(p/2, 0)
  • Direktrisa: x = -p/2
Nümunə 4.2: y² = 8x parabolasının fokusunu və direktrisasını tapaq.

Həll:
2p = 8, p = 4
Fokus: F(2, 0)
Direktrisa: x = -2

5. FƏZADA MÜSTƏVİ VƏ DÜZ XƏTT TƏNLİKLƏRİ

5.1. Fəzada Müstəvi Tənliyi

Fəzada müstəvi ümumi tənliyi:

Ax + By + Cz + D = 0

Burada n = (A, B, C) müstəvinin normal vektorudur.

Müstəvi tənliyinin xüsusi halları:

  • D = 0: müstəvi başlanğıcdan keçir
  • A = 0: müstəvi Ox oxuna paraleldir
  • B = 0: müstəvi Oy oxuna paraleldir
  • C = 0: müstəvi Oz oxuna paraleldir

5.2. Nöqtədən Keçən Müstəvi

M₀(x₀, y₀, z₀) nöqtəsindən keçən və n = (A, B, C) normal vektorlu müstəvi:

A(x - x₀) + B(y - y₀) + C(z - z₀) = 0
Nümunə 5.1: M(1, 2, 3) nöqtəsindən keçən və n = (2, -1, 3) normal vektorlu müstəvi tənliyini yazaq.

Həll:
2(x - 1) - 1(y - 2) + 3(z - 3) = 0
2x - 2 - y + 2 + 3z - 9 = 0
2x - y + 3z - 9 = 0

5.3. Fəzada Düz Xətt Tənliyi

Fəzada düz xətt iki müstəvinin kəsişməsi kimi verilir:

{A₁x + B₁y + C₁z + D₁ = 0
{A₂x + B₂y + C₂z + D₂ = 0

Parametrik forma:

x = x₀ + at
y = y₀ + bt
z = z₀ + ct

Burada M₀(x₀, y₀, z₀) - düz xətt üzərində nöqtə, s = (a, b, c) - istiqamət vektoru.

Kanonik forma:

(x - x₀)/a = (y - y₀)/b = (z - z₀)/c
Nümunə 5.2: M(1, -1, 2) nöqtəsindən keçən və s = (2, 3, -1) istiqamət vektorlu düz xətt tənliyini yazaq.

Həll:
Kanonik forma: (x-1)/2 = (y+1)/3 = (z-2)/(-1)
Parametrik forma:
x = 1 + 2t
y = -1 + 3t
z = 2 - t

5.4. Düz Xətt və Müstəvi Arasında Bucaq

Düz xətt istiqamət vektoru s = (a, b, c) və müstəvi normal vektoru n = (A, B, C) arasında bucaq:

sin φ = |aA + bB + cC| / (√(a²+b²+c²) · √(A²+B²+C²))

5.5. İki Müstəvi Arasında Bucaq

Normal vektorları n₁ = (A₁, B₁, C₁) və n₂ = (A₂, B₂, C₂) olan iki müstəvi arasında bucaq:

cos φ = |A₁A₂ + B₁B₂ + C₁C₂| / (√(A₁²+B₁²+C₁²) · √(A₂²+B₂²+C₂²))

1.13. Kompleks Ədədlərin Həndəsi Tətbiqləri

Kompleks ədədlər həndəsi çevrilmələri təsvir etmək üçün güclü vasitədir. Kompleks müstəvisində hər bir kompleks ədəd z bir nöqtəni və ya vektoru təmsil edir.

Paralel Köçürmə (Translasiya):

w = z + z₀

Bu çevrilmə z nöqtəsini z₀ vektoru qədər köçürür.

Fırlatma (Rotasiya):

w = z · e^(iθ)

Bu çevrilmə z nöqtəsini başlanğıc ətrafında θ bucağı qədər fırladır.

Miqyaslama (Skalyarlama):

w = k · z (k > 0)

Bu çevrilmə z nöqtəsini başlanğıcdan k dəfə uzaqlaşdırır və ya yaxınlaşdırır.

Nümunə: z = 1 + i nöqtəsini 45° fırladın və 2 dəfə böyüdün.

Həll:
Fırlatma: w₁ = (1 + i) · e^(i·π/4) = (1 + i) · (cos 45° + i sin 45°)
= (1 + i)(√2/2 + i√2/2) = √2i
Miqyaslama: w = 2 · √2i = 2√2i

1.14. Kompleks Ədədlərdə Bərabərsizliklər

Üçbucaq Bərabərsizliyi:

|z₁ + z₂| ≤ |z₁| + |z₂|

Tərs Üçbucaq Bərabərsizliyi:

||z₁| - |z₂|| ≤ |z₁ - z₂|

Koshi-Şvarts Bərabərsizliyi:

|z₁z̄₂ + z₂z̄₁| ≤ 2|z₁||z₂|

1.15. Kompleks Ədədlərin Tarixi və İnkişafı

Kompleks ədədlərin inkişafı riyaziyyat tarixində maraqlı bir səhifədir:

  • XVI əsr: İtalyan riyaziyyatçıları Cardano və Tartaglia kubik tənliklərin həllində mənfi ədədlərin kökü ilə qarşılaşdılar
  • 1572: Rafael Bombelli kompleks ədədlər üzərində əməllərin qaydalarını sistemləşdirdi
  • 1777: Leonhard Euler i simvolunu təqdim etdi
  • 1799: Carl Friedrich Gauss Cəbrin Əsas Teoremini isbat etdi
  • 1806: Jean-Robert Argand kompleks müstəvisinin həndəsi təsvirini verdi
  • 1831: Gauss "kompleks ədəd" terminini təklif etdi

1.16. Nəticə və Xülasə

Kompleks ədədlər riyaziyyatın ən fundamental və gözəl nəzəriyyələrindən biridir. Onlar təkcə riyazi maraq doğurmur, həm də fizika, mühəndislik, elektrotexnika, kvant mexanikası və bir çox digər sahələrdə praktik tətbiqlərə malikdir.

Kompleks ədədlər nəzəriyyəsinin əsas nailiyyətləri:

  • Həqiqi ədədlər sisteminin təbii genişlənməsi
  • Bütün polinom tənliklərin həlli
  • Triqqonometrik və eksponensial funksiyaların birləşdirilməsi
  • Həndəsi çevrilmələrin sadə təsviri
  • Fiziki proseslərin modelləşdirilməsi

2.5. Homojen Diferensial Tənliklər

Homojen diferensial tənlik aşağıdakı formada olan tənlikdir:

dy/dx = f(y/x)

Həll metodu:

  1. v = y/x əvəzləməsi aparırıq, buradan y = vx
  2. dy/dx = v + x(dv/dx) alırıq
  3. Dəyişənlərə ayrılan tənlik alırıq
Nümunə 2.3: (x² + y²)dx - 2xy dy = 0 tənliyini həll edin.

Həll:
dy/dx = (x² + y²)/(2xy)
v = y/x əvəzləməsi: y = vx, dy/dx = v + x(dv/dx)
v + x(dv/dx) = (x² + v²x²)/(2vx²) = (1 + v²)/(2v)
x(dv/dx) = (1 + v²)/(2v) - v = (1 - v²)/(2v)
2v dv/(1 - v²) = dx/x
-ln|1 - v²| = ln|x| + C
1 - v² = C₁/x
1 - y²/x² = C₁/x
x² - y² = C₁x

2.6. Tam Diferensial Tənliklər

M(x,y)dx + N(x,y)dy = 0 tənliyi tam diferensial tənlik adlanır, əgər:

∂M/∂y = ∂N/∂x

Bu halda tənliyin həlli:

U(x,y) = C
burada ∂U/∂x = M və ∂U/∂y = N

2.7. İnteqrallayıcı Vuruq Metodu

Bəzi hallarda tam olmayan diferensial tənliyi μ(x,y) funksiyasına vuraraq tam tənliyə gətirmək olar. Bu funksiya inteqrallayıcı vuruq adlanır.

Xüsusi hallar:

  • Əgər (1/N)(∂M/∂y - ∂N/∂x) = f(x) olarsa, μ = e^(∫f(x)dx)
  • Əgər (1/M)(∂N/∂x - ∂M/∂y) = g(y) olarsa, μ = e^(∫g(y)dy)

2.8. Riccati Tənliyi

Riccati tənliyi aşağıdakı formadadır:

y' = P(x)y² + Q(x)y + R(x)

Bu tənliyin ümumi həlli yoxdur, lakin bir xüsusi həll məlum olarsa, ümumi həll tapıla bilər.

2.9. Diferensial Tənliklərin Tətbiqləri

1. Fizikada:

  • Mexaniki hərəkət tənlikləri
  • Elektrik dövrələri
  • İstilik keçiriciliyi
  • Dalğa yayılması

2. Biologiyada:

  • Populyasiya dinamikası
  • Epidemiya modelləri
  • Fermentativ reaksiyalar

3. İqtisadiyyatda:

  • İqtisadi artım modelləri
  • Bazar tarazlığı
  • İnvestisiya strategiyaları
Praktik Tətbiq - Radioaktiv Parçalanma:

Radioaktiv maddənin parçalanma sürəti onun miqdarı ilə mütənasibdir:
dN/dt = -λN

Həll: N(t) = N₀e^(-λt)

Burada N₀ - başlanğıc miqdar, λ - parçalanma sabiti, t - vaxt.

2.10. Xətti Diferensial Tənliklərin Sistemi

İki naməlum funksiyalı xətti sistem:

dx/dt = a₁₁x + a₁₂y + f₁(t)
dy/dt = a₂₁x + a₂₂y + f₂(t)

Homojen Sistemin Həlli:

f₁(t) = f₂(t) = 0 olduqda, həll e^(λt) formasında axtarılır.

2.11. Laplace Çevrilməsi ilə Həll

Laplace çevrilməsi diferensial tənlikləri cəbri tənliklərə çevirir:

L{f(t)} = F(s) = ∫₀^∞ e^(-st)f(t)dt

Əsas Xassələr:

  • L{f'(t)} = sF(s) - f(0)
  • L{f''(t)} = s²F(s) - sf(0) - f'(0)
  • L{e^(at)f(t)} = F(s-a)

2.12. Nəticə

Diferensial tənliklər təbiət qanunlarını riyazi dildə ifadə etməyə imkan verir. Onlar fizika, kimya, biologiya, iqtisadiyyat və mühəndisliyin əsasını təşkil edir. Bir tərtibli diferensial tənliklərin həll metodlarını öyrənmək daha mürəkkəb tənlikləri anlamaq üçün zəruridir.

3.5. Xətti Tənliklər Sistemi

n naməlumlu n xətti tənlik sistemi:

a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ = b₂
...
aₙ₁x₁ + aₙ₂x₂ + ... + aₙₙxₙ = bₙ

Matris formasında: AX = B

Kramer Metodu:

Əgər det(A) ≠ 0 olarsa:

xᵢ = det(Aᵢ) / det(A)

Burada Aᵢ - A matrisinin i-ci sütununu B sütunu ilə əvəz etməklə alınan matrisdir.

Nümunə 3.3: Kramer metodu ilə həll edin:
2x + y = 5
x - y = 1

Həll:
A = |2 1|, det(A) = -2 - 1 = -3
    |1 -1|

A₁ = |5 1|, det(A₁) = -5 - 1 = -6
     |1 -1|

A₂ = |2 5|, det(A₂) = 2 - 5 = -3
     |1 1|

x = -6/(-3) = 2
y = -3/(-3) = 1

3.6. Qauss Metodu

Genişləndirilmiş matrisi pilləli formaya gətirməklə həll tapılır:

  1. Genişləndirilmiş matris [A|B] yaradılır
  2. Elementar sətir əməlləri ilə pilləli forma alınır
  3. Geriyə əvəzetmə ilə həll tapılır

3.7. Matris Tənlikləri

AX = B matris tənliyinin həlli (det(A) ≠ 0 olduqda):

X = A⁻¹B

3.8. Xarakteristik Tənlik və Xüsusi Qiymətlər

A matrisinin xüsusi qiymətləri λ aşağıdakı tənliyin köklərdir:

det(A - λI) = 0

Bu tənlik xarakteristik tənlik adlanır.

Nümunə 3.4: A = |3 1| matrisinin xüsusi qiymətlərini tapın.
                         |1 3|

Həll:
det(A - λI) = |3-λ 1 | = (3-λ)² - 1 = 0
                  | 1 3-λ|
9 - 6λ + λ² - 1 = 0
λ² - 6λ + 8 = 0
(λ - 4)(λ - 2) = 0
λ₁ = 4, λ₂ = 2

3.9. Matrislərin Diaqonallaşdırılması

A matrisi diaqonallaşdırıla bilər, əgər n xətti asılı olmayan xüsusi vektoru varsa:

A = PDP⁻¹

Burada D - diaqonal matris (xüsusi qiymətlərlə), P - xüsusi vektorlardan ibarət matris.

3.10. Matrislərin Tətbiqləri

1. Kompüter Qrafikası:

  • 2D və 3D çevrilmələr
  • Fırlatma, miqyaslama, əyilmə
  • Proyeksiya matrisləri

2. Kriptoqrafiya:

  • Hill şifrələməsi
  • Açıq açar kriptoqrafiyası

3. İqtisadiyyat:

  • Leontief input-output modeli
  • Markov zəncirləri
  • Portfel optimizasiyası

4. Fizika:

  • Kvant mexanikası operatorları
  • Gərginlik və deformasiya tensorları
  • Fırlanma matrisləri
Praktik Tətbiq - 2D Fırlatma:

Nöqtəni (x, y) başlanğıc ətrafında θ bucağı qədər fırlatmaq üçün:

|x'| = |cos θ -sin θ| |x|
|y'|   |sin θ cos θ| |y|

Məsələn, (1, 0) nöqtəsini 90° fırlatmaq:
|x'| = |0 -1| |1| = | 0|
|y'|   |1 0| |0|   | 1|

Nəticə: (0, 1)

3.11. Nəticə

Matris və determinant nəzəriyyəsi xətti cəbrin əsasını təşkil edir. Bu nəzəriyyə təkcə riyaziyyatda deyil, həm də fizika, mühəndislik, kompüter elmləri, iqtisadiyyat və bir çox digər sahələrdə geniş tətbiq olunur.

4.5. Əyrilərin Ümumi Tənliyi

İki tərtibli əyrinin ümumi tənliyi:

Ax² + Bxy + Cy² + Dx + Ey + F = 0

Əyrinin növü diskriminantla müəyyən olunur:

  • B² - 4AC < 0: Ellips (və ya çevrə)
  • B² - 4AC = 0: Parabola
  • B² - 4AC > 0: Hiperbola

4.6. Çevrənin Ətraflı Xassələri

Çevrənin Tənlik Formaları:

Standart forma: (x - a)² + (y - b)² = r²
Ümumi forma: x² + y² + Dx + Ey + F = 0

Çevrənin Parametrik Tənliyi:

x = a + r cos t
y = b + r sin t
0 ≤ t ≤ 2π

Çevrə ilə Düz Xəttin Qarşılıqlı Vəziyyəti:

  • d < r: İki kəsişmə nöqtəsi
  • d = r: Toxunma (bir nöqtə)
  • d > r: Kəsişmə yoxdur

Burada d - mərkəzdən düz xəttə qədər məsafə.

Nümunə 4.3: x² + y² - 4x + 6y - 3 = 0 çevrəsinin mərkəzini və radiusunu tapın.

Həll:
Tam kvadrata gətirək:
(x² - 4x + 4) + (y² + 6y + 9) = 3 + 4 + 9
(x - 2)² + (y + 3)² = 16

Mərkəz: (2, -3)
Radius: r = 4

4.7. Ellipsin Ətraflı Xassələri

Ellipsin Parametrik Tənliyi:

x = a cos t
y = b sin t
0 ≤ t ≤ 2π

Ellipsin Əsas Elementləri:

  • Fokuslar: F₁(-c, 0), F₂(c, 0), c = √(a² - b²)
  • Böyük ox: 2a
  • Kiçik ox: 2b
  • Ekssentrikllik: e = c/a, 0 < e < 1
  • Direktrisalar: x = ±a/e

Ellipsin Fokal Xassəsi:

Ellips üzərindəki istənilən M nöqtəsi üçün:

MF₁ + MF₂ = 2a
Nümunə 4.4: 9x² + 16y² = 144 ellipsinin bütün parametrlərini tapın.

Həll:
Standart formaya gətirək: x²/16 + y²/9 = 1
a² = 16, a = 4
b² = 9, b = 3
c = √(16 - 9) = √7
e = √7/4 ≈ 0.66
Fokuslar: F₁(-√7, 0), F₂(√7, 0)
Direktrisalar: x = ±16/√7

4.8. Hiperbolanın Ətraflı Xassələri

Hiperbolanın Parametrik Tənliyi:

x = a sec t = a/cos t
y = b tan t

Hiperbolanın Əsas Elementləri:

  • Fokuslar: F₁(-c, 0), F₂(c, 0), c = √(a² + b²)
  • Həqiqi ox: 2a
  • Xəyali ox: 2b
  • Ekssentrikllik: e = c/a, e > 1
  • Asimptotlar: y = ±(b/a)x

Hiperbolanın Fokal Xassəsi:

Hiperbola üzərindəki istənilən M nöqtəsi üçün:

|MF₁ - MF₂| = 2a

4.9. Parabolanın Ətraflı Xassələri

Parabolanın Müxtəlif Formaları:

Tənlik Fokus Direktrisa
y² = 2px F(p/2, 0) x = -p/2
y² = -2px F(-p/2, 0) x = p/2
x² = 2py F(0, p/2) y = -p/2
x² = -2py F(0, -p/2) y = p/2

Parabolanın Fokal Xassəsi:

Parabola üzərindəki istənilən M nöqtəsi üçün fokusdan və direktrisadan məsafələr bərabərdir.

4.10. Əyrilərin Tətbiqləri

1. Astronomiya:

  • Planetlərin orbitləri - ellips
  • Kometlərin yolları - parabola və ya hiperbola

2. Mühəndislik:

  • Körpü qövsləri - parabola
  • Reflektor antennalar - parabola
  • Soyuducu qüllələr - hiperbola

3. Optika:

  • Teleskop güzgüləri - parabola

Parabolik antennalar siqnalları fokusda cəmləyir. Əgər antena y² = 8x parabolası formasındadırsa, qəbuledici fokusda yerləşməlidir.

2p = 8, p = 4
Fokus: F(2, 0)

Qəbuledici (2, 0) nöqtəsində yerləşdirilməlidir.

5.6. Nöqtədən Müstəviyə Qədər Məsafə

M₀(x₀, y₀, z₀) nöqtəsindən Ax + By + Cz + D = 0 müstəvisinə qədər məsafə:

d = |Ax₀ + By₀ + Cz₀ + D| / √(A² + B² + C²)
Nümunə 5.3: M(1, 2, 3) nöqtəsindən 2x - y + 2z - 3 = 0 müstəvisinə qədər məsafəni tapın.

Həll:
d = |2·1 - 1·2 + 2·3 - 3| / √(4 + 1 + 4)
= |2 - 2 + 6 - 3| / 3
= 3/3 = 1

5.7. İki Müstəvinin Qarşılıqlı Vəziyyəti

İki müstəvi:

A₁x + B₁y + C₁z + D₁ = 0
A₂x + B₂y + C₂z + D₂ = 0

Vəziyyətlər:

  • Paralel: A₁/A₂ = B₁/B₂ = C₁/C₂ ≠ D₁/D₂
  • Üst-üstə düşür: A₁/A₂ = B₁/B₂ = C₁/C₂ = D₁/D₂
  • Kəsişir: A₁/A₂ ≠ B₁/B₂ və ya B₁/B₂ ≠ C₁/C₂
  • Perpendikulyar: A₁A₂ + B₁B₂ + C₁C₂ = 0

5.8. Düz Xətt və Müstəvinin Kəsişməsi

Düz xətt (x-x₀)/l = (y-y₀)/m = (z-z₀)/n və müstəvi Ax + By + Cz + D = 0 kəsişmə nöqtəsini tapmaq üçün:

  1. Düz xəttin parametrik tənliyini yazırıq
  2. Müstəvi tənliyinə əvəz edirik
  3. t parametrini tapırıq
  4. Kəsişmə nöqtəsinin koordinatlarını hesablayırıq
Nümunə 5.4: (x-1)/2 = (y+1)/1 = (z-2)/3 düz xətti ilə x + y + z - 6 = 0 müstəvisinin kəsişmə nöqtəsini tapın.

Həll:
Parametrik forma: x = 1 + 2t, y = -1 + t, z = 2 + 3t
Müstəviyə əvəz edək:
(1 + 2t) + (-1 + t) + (2 + 3t) - 6 = 0
2 + 6t - 6 = 0
6t = 4
t = 2/3

x = 1 + 2·(2/3) = 7/3
y = -1 + 2/3 = -1/3
z = 2 + 3·(2/3) = 4

Kəsişmə nöqtəsi: (7/3, -1/3, 4)

5.9. İki Düz Xəttin Qarşılıqlı Vəziyyəti

Fəzada iki düz xətt:

  • Kəsişir: Bir nöqtədə kəsişirlər
  • Paralel: İstiqamət vektorları kollineardır
  • Çarpaz: Bir müstəvidə yerləşmir, kəsişmir

5.10. Sfera Tənliyi

Mərkəzi C(a, b, c) və radiusu r olan sferanın tənliyi:

(x - a)² + (y - b)² + (z - c)² = r²

Ümumi forma:

x² + y² + z² + Dx + Ey + Fz + G = 0

5.11. Silindrik və Sferik Koordinatlar

Silindrik Koordinatlar:

x = ρ cos φ
y = ρ sin φ
z = z

Sferik Koordinatlar:

x = r sin θ cos φ
y = r sin θ sin φ
z = r cos θ
